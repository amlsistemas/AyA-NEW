El atributo de evento onclick reacciona cuando un usuario hace clic. Puede cambiar el color de fondo del cuadro, cambiar la fuente del texto, contar cuántas veces hace clic el usuario y más. Haga clic aquí en este cuadro para ver qué puede hacer con el atributo onclick y un poco de JavaScript. Cuando combinas JavaScript con HTML y CSS, ¡puedes hacer que tus páginas web cobren vida!
